# dunatura_newsletter

A simple HTML newsletter for [dunatura](https://www.dunatura.com/), built with [MJML](https://mjml.io/).

Result: index.html file

## How to use
1. [Install MJML](https://mjml.io/download)
2. Run ```mjml -r index.mjml -o index.html``` to compile to a html file

## Responsiveness
- MJML handles most of the responsiveness if their guidelines are followed
- ⚠️ Font sizes must be manually handled (media queries)
- ⚠️ Custom fonts are not supported in many email clients, specify fallback fonts!
